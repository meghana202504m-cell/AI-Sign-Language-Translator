# AI Sign Language Translator

A real-time AI-based Sign Language Translator developed using **Python**, **OpenCV**, and **MediaPipe**.  
This project detects hand gestures through a webcam and translates them into sign language characters in real time.

---

## 📌 Project Overview

Communication between sign language users and non-signers can be challenging.  
This project aims to bridge that communication gap using computer vision and AI-based hand gesture recognition.

The system captures live video from the webcam, detects hand landmarks using MediaPipe, and recognizes gestures dynamically.

---

## 🚀 Features

✅ Real-time hand tracking  
✅ Sign gesture recognition using webcam  
✅ Lightweight and fast implementation  
✅ Built using OpenCV + MediaPipe  
✅ Beginner-friendly AI project  
✅ Works without expensive hardware  
✅ Easy to extend for full A–Z sign detection
